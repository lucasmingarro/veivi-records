Dennis - Responsive Next Js Personal Portfolio Template

Introduction

Dennis is a responsive Next Js personal portfolio template. It’s a modern, creative, clean, professional, attractive personal template. It is a beautifully handcrafted, pixel-perfect React Next.js personal template based on the React 18.x, Next 13.x & Tailwind CSS v3.3.3. Dennis comes with 5 home demo pages. All files are organized we believe it will be easy to use and edit them. This template is well-organized and very easy to customize. It’s well documented which will help you to get your site running easily and fast. You can customize it very easy to fulfill your business needs.

We have added dark version and RTL vesion.

Please feel free to get back to me in case if you are having any question or feedback.

How can you change the font-family?

Please follow the step by step process

Step 1

Import your font family in layout.js (path: Dennis_Nextjs/src/app/layout.js) file, or replace your font name in layout.js (path: Dennis_Nextjs/src/app/layout.js) file.

import { Poppins } from 'next/font/google' - Themes primary font family is Poppins
Step 2

Pass your font family in layout.js (path: Dennis_Nextjs/src/app/layout.js) file, or replace your font name in layout.js (path: Dennis_Nextjs/src/app/layout.js) file.

const poppins = Poppins({
  subsets: ['latin'],
  weight: ["400", "500", "600", "700"],
  display: "swap",
  variable: "--font-poppins",
})

- Themes primary font family is Poppins
Step 3

Declare your font family variables in tailwind.config.js file, or replace your font name in tailwind.config.js file.

fontFamily: {
'poppins': "var(--font-poppins)"
},
- Themes primary font family is Poppins
